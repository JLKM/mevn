# Scms

    a simple blog content management system based on vue-server-rendering

Demo: http://blog.scitnet.com

#Technology Stacks

Front-end:Vuejs/Vuex/Vue-Router/Bootstrap/Jquery

Back-end: Nuxtjs/Express

Database:Mongodb
